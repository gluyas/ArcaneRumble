# Snow and Iron

### Controls
Players share the keyboard, taking one side each

| Action | Player 1 | Player 2 |
| :----: | :------: | :-------:|
| Select Unit | `1` `2` `3` | `8` `9` `0` |
| Move / Rotate | `W` `A` `S` `D` | `I` `J` `K` `L` |
| Swizzle | `Q` | `O` |
| Deploy |  `E` | `U` |
